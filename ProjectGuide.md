# Project Overview

## Goal

참고 이미지(koreapuzzle.svg)의 외곽선을 그대로 유지한 상태로, 한반도 지도를 정확히 100개의 조각으로 자유롭게 분할한 웹 페이지를 제작한다.

---

# Reference Image

반드시 제공된 참고 이미지 koreapuzzle.svg를 기준으로 작업한다.

구현 시 다음 요소를 참고 이미지와 최대한 동일하게 맞춘다.

한반도 전체 외곽선: 원본 이미지의 형태를 절대 수정하지 않고 그대로 사용한다.

전체적인 비율과 배치: 원본 이미지의 비율을 왜곡 없이 웹 화면에 표시한다.

---

# Map Requirements

## Shape

- 한반도 모양은 참고 이미지의 형태를 그대로 사용한다.
- 임의의 다른 한반도 지도 이미지를 사용하지 않는다.

---

# Interaction

사용자가 각 퍼즐 조각을 클릭하면 다음 동작을 수행한다.

## Color Change

클릭한 조각만 내부 색상이 변경된다.
조각마다 서로 다른 무작위 색상(Random Color)이 적용되도록 한다.
이미 클릭된 조각은 다시 클릭할 수 없도록 처리한다.

# Technical Requirements

HTML, CSS, JavaScript로 구현한다.

SVG 활용: 100개의 퍼즐 조각을 개별 PNG 이미지로 나누지 않고, 단일 SVG 파일 내에 각각 독립적인 <path> 태그로 구현한다.

정밀한 클릭 영역: SVG의 특성을 활용하여, 투명한 사각형 여백의 간섭 없이 각 조각의 불규칙한 실제 윤곽선 안쪽 영역만 클릭 이벤트를 감지하도록 만든다.

코드 구조는 유지보수가 용이하고 가독성 있게 작성한다.

# Important Notes

- 가장 중요한 것은 참고 이미지의 모양과 분할 방식을 그대로 따르는 것이다.
- 한반도 외곽선과 내부 분리선이 참고 이미지와 크게 달라지면 안 된다.
