# 한반도 땅따먹기 게임 추가 구현 명세

## 1. 개요

10개 팀이 한반도 지도에 배치된 땅 조각을 차지하며 점수를 획득하는 방식의 게임이다.

각 땅 조각을 클릭하면 해당 땅에 배정된 점수가 팝업으로 표시되며, 진행자가 점수를 받을 팀을 선택하면 다음 처리가 이루어진다.

- 선택한 땅 조각이 해당 팀의 색상으로 변경된다.
- 해당 팀의 총점에 땅 점수가 합산된다.
- 지도 오른쪽의 팀별 점수 순위가 즉시 갱신된다.

---

## 2. 주요 화면 구성

전체 화면은 다음 두 영역으로 구성한다.

### 2.1 지도 영역

- 화면 왼쪽에 한반도 지도를 표시한다.
- 지도는 여러 개의 독립적인 땅 조각으로 구성한다.
- 각 땅 조각은 개별적으로 클릭할 수 있어야 한다.
- 아직 점령되지 않은 땅은 흰색 또는 기본 색상으로 표시한다.
- 점령된 땅은 해당 팀에 사전 배정된 색상으로 표시한다.

### 2.2 점수 순위 영역

- 지도 오른쪽에 1조부터 10조까지의 현재 점수를 표시한다.
- 각 팀 항목에는 다음 정보를 함께 표시한다.
  - 현재 순위
  - 조 번호
  - 팀 색상
  - 현재 총점
- 점수가 높은 팀부터 내림차순으로 정렬한다.
- 팀 점수가 변경될 때마다 순위를 즉시 다시 계산한다.

예시:

```text
1위  ■ 3조   540점
2위  ■ 7조   490점
3위  ■ 1조   430점
...
```

동점인 경우에는 동일 순위로 표시하거나, 조 번호가 작은 팀을 먼저 표시한다.  
정렬 규칙은 구현 시 하나로 통일한다.

---

## 3. 땅 조각 클릭 동작

### 3.1 클릭 가능 조건

- 아직 어떤 팀에도 배정되지 않은 땅 조각만 클릭할 수 있다.
- 이미 점령된 땅 조각을 클릭했을 때는 새 점수를 다시 부여하지 않는다.
- 이미 점령된 땅을 클릭할 경우 다음 중 한 가지 방식으로 처리한다.
  - 아무 동작도 하지 않음
  - `이미 점령된 땅입니다.` 안내 표시
  - 기존 점령 정보를 보여주는 별도 팝업 표시

기본 구현은 `이미 점령된 땅입니다.` 안내 표시 방식으로 한다.

### 3.2 점수 생성

땅 조각을 처음 클릭한 시점에 해당 땅의 점수를 무작위로 생성한다.

- 최소 점수: 10점
- 최대 점수: 200점
- 점수 단위: 10점

가능한 점수:

```text
10, 20, 30, ..., 190, 200
```

점수 생성 예시:

```javascript
const score = (Math.floor(Math.random() * 20) + 1) * 10;
```

생성된 점수는 팀 선택이 완료될 때까지 유지한다.  
팝업을 닫았다가 같은 땅을 다시 클릭해도 점수를 다시 추첨하지 않는 것을 기본 동작으로 한다.

---

## 4. 점수 배정 팝업

땅 조각을 클릭하면 화면 중앙에 팝업을 표시한다.

### 4.1 팝업 구성

팝업은 다음 요소로 구성한다.

1. 안내 문구
2. 해당 땅에 배정된 점수
3. 팀 선택 버튼
4. 닫기 버튼

예시:

```text
┌────────────────────────────┐
│        획득 가능 점수       │
│                            │
│           120점            │
│                            │
│  [1조] [2조] [3조] [4조] [5조] │
│  [6조] [7조] [8조] [9조] [10조]│
│                            │
│            [닫기]           │
└────────────────────────────┘
```

### 4.2 점수 표시

- 팝업 중앙에 점수를 크게 표시한다.
- 단위는 `점`으로 표시한다.
- 예: `120점`

### 4.3 팀 선택 버튼

- 팝업 하단에 1조부터 10조까지의 팀 선택 버튼을 배치한다.
- 버튼은 2행 5열 구조로 배치한다.

배치 예시:

| 1조 | 2조 | 3조 | 4조 | 5조 |
|---|---|---|---|---|
| 6조 | 7조 | 8조 | 9조 | 10조 |

- 각 버튼에는 해당 팀의 사전 지정 색상을 함께 표시한다.
- 버튼 전체 배경을 팀 색상으로 표시하거나, 버튼 내부에 색상 원형 아이콘을 표시한다.
- 색상이 밝아 글자가 잘 보이지 않는 경우 검은색 글자를 사용한다.
- 색상이 어두운 경우 흰색 글자를 사용한다.

---

## 5. 팀 선택 완료 처리

팀 버튼을 클릭하면 다음 작업을 순서대로 실행한다.

1. 선택한 팀을 현재 땅의 소유 팀으로 저장한다.
2. 현재 땅에 배정된 점수를 해당 팀의 총점에 더한다.
3. 땅 조각의 색상을 해당 팀의 색상으로 변경한다.
4. 지도 오른쪽의 점수 및 순위를 갱신한다.
5. 팝업을 닫는다.
6. 해당 땅을 다시 선택할 수 없도록 비활성화한다.

처리 흐름 예시:

```javascript
function assignLandToTeam(landId, teamId) {
  const land = lands[landId];
  const team = teams[teamId];

  if (land.ownerTeamId !== null) {
    return;
  }

  land.ownerTeamId = teamId;
  team.score += land.score;

  updateLandColor(landId, team.color);
  updateRanking();
  closeScorePopup();
}
```

## 6. 팀 정보

팀은 총 10개이며, 각 팀에는 게임 시작 전 고유 색상을 배정한다.

각 팀이 가져야 할 데이터:

```javascript
{
  id: 1,
  name: "1조",
  color: "#색상코드",
  score: 0
}
```

팀 데이터 예시:

const teams = [

## 7. 땅 조각 데이터

각 땅 조각은 최소한 다음 정보를 가져야 한다.

```javascript
{
  id: "land-001",
  score: null,
  ownerTeamId: null
}
```

### 필드 설명

| 필드 | 설명 |
|---|---|
| `id` | 땅 조각의 고유 식별자 |
| `score` | 해당 땅에 배정된 점수 |
| `ownerTeamId` | 땅을 차지한 팀 ID |
| `null` | 아직 값이 정해지지 않은 상태 |

땅 조각을 처음 클릭했을 때만 `score`를 생성한다.

```javascript
function getOrCreateLandScore(land) {
  if (land.score === null) {
    land.score = (Math.floor(Math.random() * 20) + 1) * 10;
  }

  return land.score;
}
```

## 8. 점수 순위 계산

팀 순위는 각 팀의 현재 총점을 기준으로 계산한다.

```javascript
function getTeamRanking(teams) {
  return [...teams].sort((a, b) => {
    if (b.score !== a.score) {
      return b.score - a.score;
    }

    return a.id - b.id;
  });
}
```

정렬 기준:

1. 총점이 높은 팀 우선
2. 총점이 같다면 조 번호가 작은 팀 우선

점수가 변경될 때마다 점수판 전체를 다시 렌더링한다.

---

## 9. 팝업 취소 처리

팀을 선택하지 않고 팝업을 닫을 수 있어야 한다.

팝업을 닫았을 때:

- 땅은 점령되지 않은 상태로 유지한다.
- 어느 팀에도 점수를 부여하지 않는다.
- 생성된 땅 점수는 유지한다.
- 같은 땅을 다시 클릭하면 이전과 동일한 점수를 표시한다.

팝업 닫기 방식:

- 팝업 내부의 `닫기` 버튼
- 팝업 바깥의 어두운 배경 클릭
- 키보드 `ESC` 입력

게임 진행 중 실수 방지를 위해 배경 클릭과 `ESC` 닫기는 비활성화하고, `닫기` 버튼만 허용하는 방식도 가능하다.

## 10. 확인 단계 권장

팀 버튼을 잘못 누르는 상황을 방지하기 위해 최종 확인 단계를 추가할 수 있다.

예시:

```text
120점을 3조에 부여하시겠습니까?

[취소] [확인]
```

`확인`을 눌렀을 때만 실제 점수와 땅 소유권을 반영한다.

이 기능은 선택 사항이지만, 실제 게임 진행 시 오입력 방지를 위해 사용하는 것을 권장한다.

---

## 11. 상태 저장

페이지를 새로고침해도 게임 상태가 유지되어야 한다면 다음 정보를 브라우저 저장소에 저장한다.

- 각 땅의 점수
- 각 땅의 소유 팀
- 각 팀의 현재 총점

저장 예시:

```javascript
localStorage.setItem("landGameState", JSON.stringify({
  teams,
  lands
}));
```

페이지 실행 시 저장된 상태가 있으면 불러온다.

```javascript
const savedState = localStorage.getItem("landGameState");

if (savedState) {
  const gameState = JSON.parse(savedState);
}
```

---

## 12. 게임 초기화

점수판 오른쪽 하단에 `게임 초기화` 버튼을 추가한다.

초기화 버튼 클릭 시 확인 팝업을 표시한다.

```text
모든 땅과 점수를 초기화하시겠습니까?

[취소] [초기화]
```

초기화 완료 후:

- 모든 팀 점수를 0점으로 변경
- 모든 땅의 소유 팀 제거
- 모든 땅의 점수 제거
- 모든 땅을 기본 색상으로 변경
- 저장된 게임 상태 삭제
- 점수 순위 초기화

## 13. 최종 동작 흐름

```text
땅 조각 클릭
    ↓
해당 땅의 점수 확인
    ↓
점수 팝업 표시
    ↓
1조~10조 중 팀 선택
    ↓
선택 내용 최종 확인
    ↓
땅 색상을 팀 색상으로 변경
    ↓
팀 점수에 땅 점수 합산
    ↓
오른쪽 팀 순위 갱신
    ↓
게임 상태 저장
```

## 14. 필수 구현 체크리스트

- [ ] 각 땅 조각을 개별적으로 클릭할 수 있다.
- [ ] 땅 클릭 시 팝업이 표시된다.
- [ ] 팝업 중앙에 10~200점 사이의 무작위 점수가 표시된다.
- [ ] 점수는 10점 단위로 생성된다.
- [ ] 팝업 하단에 1조~10조 버튼이 표시된다.
- [ ] 팀 버튼은 2행 5열로 배치된다.
- [ ] 각 팀은 서로 다른 사전 지정 색상을 가진다.
- [ ] 팀 선택 시 땅이 해당 팀 색상으로 변경된다.
- [ ] 팀 선택 시 점수가 해당 팀의 총점에 합산된다.
- [ ] 이미 점령된 땅은 다시 점령할 수 없다.
- [ ] 지도 오른쪽에 팀별 현재 점수가 표시된다.
- [ ] 점수가 높은 순서대로 팀 순위가 표시된다.
- [ ] 순위 목록에 조 번호와 팀 색상이 함께 표시된다.
- [ ] 점수 변경 시 순위가 즉시 갱신된다.
- [ ] 새로고침 후에도 상태가 유지된다.
- [ ] 전체 게임을 초기화할 수 있다.
